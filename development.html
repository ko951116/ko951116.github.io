Maysonry Layout Photo Album

I like taking pictures. Especially, I like taking pictures of beautiful scenery. I always take pictures when I look at unique things or signs. I made an online photo album focusing on various pictures I took while traveling, especially landscape photos.

I usually use the Pinterest site, and I like the way Pinterest pictures are arranged. I found out that this was called Masonry Layout, and I used this layout to make my photo album. The library was not used to implement the Masonry layout.

If the mouse is placed on top of the picture, it had the effect of darkening only the center part like a frame to maintain the image of the photo album. I added fun by creating a dark mode that changes the overall feeling of darkness.

Shortest path in a Maze

I implemented a maze escape project using BFS algorithm called Breadth-first search. Various algorithms such as DFS, BFS, A* algorithms can be used for maze escape, and I applied the commonly used BFS algorithm. BFS algorithm always guarantees the shortest path, although its spatial complexity is significant.

I was able to learn more about BFS algorithm while conducting this maze escape program. The result screen shows the direction of the escape, the time taken and the result.

Moving the cube

To study OpenGL, you need to know exactly what axis and viewport is. I implemented a program that can draw and transform the cube. Press the middle button of the mouse to display menus such as translate, rotate, scailing, camera, etc., select the desired axis among the x, y, and z axes, and then press the left or right button of the keyboard to transform the cube.

CDs and print fee balance checking website

I borrowed the program CD from the department office several times, but I used to forget the date when I should return it. So I returned it late and paid a lot of late fees. Once there was even a situation where I felt embarrassed when I wanted to print out because I had an insufficient balance, so I thought it would be nice to have a website that allows students to check these two things themselves.

I made two versions of the website using JSP and Java. I used MySQL to use the database using actual data information, and various programs for development environment.

To me, this project was almost the first project to set up a problem and try to solve it. Personally, it was fun and rewarding to work to solve problems that my department friends, including myself, were uncomfortable with.
