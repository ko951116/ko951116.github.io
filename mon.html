MON
MON's story using BFS Algorithm, Front-end development, Arduino & Paper craft

Project Overview

This year, I was depressed and sometimes scared as I couldn't go out much due to the pandemic. Sometimes I felt like my ice cream in the refrigerator was disappearing, and the order of the books in my bookshelf was changed. This may be because I stayed at home for too long.. lol

I decided to have a funny idea. A small character named MON came to my house, played, and hid. Each story was set using algorithmic methods, front-end development, and Arduino.

This is a small personal project that I did in the pandemic era only to give myself a little pleasure.

The Process

STEP 01

Situation: Find the shortest path to exit the maze

Let's say you don't know which destination is Jiyoon's house, 1 or 2. Anyway, this house is closest to where Mon starts. How can MON find its destination?

BFS Algorithm

View Code

1Apply all possible operators (sucessor function) to the start node

2Apply all possible operators to all the direct successors of the start node

3Apply all possible operators to their successors till goal node found

Result

No. 2 was Jiyoon's house. MON found its destination in 18 steps.

STEP 03

Situation: MON hide in the bookshelf

MON hid on the bookshelf. Jiyoon measured the distance with ultrasonic waves to see where it hid, made a LED mood light, and put it on the bookshelf with other books.

Idea

Control LED using an ultrasonic sensor.

Craft

I used recycled paper for the craft.

Arduino

1 An object approaches the ultrasonic sensor

2Ultrasonic sensors emit ultrasonic waves on objects and change the return time in centimeters.

3Adjust the LED brightness according to distance.
